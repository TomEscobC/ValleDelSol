# Frontend: Sistema Integrado Valle del Sol (Angular)
Este componente frontend ha sido empaquetado bajo el estándar NPM. Implementa el Patrón Observador (Observer Pattern) utilizando RxJS para consumir alertas de incendios en tiempo real sin sobrecargar el servidor, garantizando alta eficiencia.

## Instrucciones de Instalación y Ejecución
1. Abre una terminal en esta ruta (`frontend-valle-sol`).
2. Instala las dependencias del `package.json` ejecutando: `npm install`
3. Levanta el servidor de desarrollo ejecutando: `ng serve`
4. Abre tu navegador en `http://localhost:4200/` para visualizar el Dashboard GIS y la App Ciudadana.
